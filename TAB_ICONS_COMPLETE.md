### **What It Does:**
This button allows users to **find trends within a specific radius** of their current location.

### **How It Works:**
1. User clicks **"Use my location"** (gets GPS coordinates)
2. User enters a **radius in km** (e.g., 5 km)
3. User clicks **"Search Nearby Trends"**
4. App shows only trends posted within that radius

- **Example**: You're in downtown and want to see what's trending within 5km
